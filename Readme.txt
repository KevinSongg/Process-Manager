Instructions to run the process manager

1. Put all files in the same folder
2. Run the command 'make' from the terminal
3. Next do ./pman in the terminal to access Plan

In Pman you can choose to run 6 commands.
\
1. bg <cmd> starts a running command in the background
2. bglist lists all the processes that are running
3. bgkill <pid> terminates the process with the pid
4. bgstop <pid> stops the process with the pid temporarily
5. bgstart <pid> resumes the process with the pid that was previously stopped
6. pstat <pid> lists the different status's of the pid
7. q allows u to leave Pman
